Memoire
OS-memory-simulation

Description

This projects aims to simulate the malloc() and free() system functions by creating a virtual memory and managing it following common system policies (paging, allocation operations: first/best/worst fit, process simulation).

The projects uses ncurses library for an intuitive graphical interface.

The goal is purely academic, the project can be used to demonstrate memory usage for new computer science students.

Requirements

  • gcc
  • Cmake>=3.13
  • make
  • ncurses
  • doxygen (if you need to modify the documentation only)

How to build

  • cmake -G "Unix Makefiles" -B/build/
  • make -C build/
